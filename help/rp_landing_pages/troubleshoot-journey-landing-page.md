# トラッキング、監視、トラブルシューティング {#section-overview}

カスタマージャーニーを効果的に管理するには、包括的なトラッキングから始まり、プロアクティブな監視、洞察に満ちたレポート、効率的なトラブルシューティングが可能になります。 Adobe Journey Optimizerは、お客様とのやり取りやシステムイベントを把握し、パフォーマンス指標を分析し、問題が発生した場合にそれを迅速に解決するための包括的なフレームワークを提供します。

メール、web、プッシュ、アプリ内、SMS など **あらゆるチャネルをまたいだ顧客インタラクション** トラッキング）では、顧客がコンテンツとどのように関わっているかを示す、開封数、クリック数、コンバージョン数、カスタムイベントをキャプチャします。

**リアルタイムレポート** システムアラートおよび配信品質指標を使用して監視、ジャーニーおよびキャンペーンのパフォーマンスを行い、トレンドの特定、成功の測定および潜在的な問題の早期発見に役立てます。

**トラブルシューティング** 詳細なエラーコード、ステップバイステップの実行ログ、ジャーニーのエラー、実行の不整合、アクションレベルの問題を解決するためのターゲットガイダンスを使用して、問題を効率的に処理します。

このページには、トラッキングの実装、ジャーニーの正常性の監視、主要なレポートインサイトへのアクセス、カスタマージャーニーのすべての段階にわたる問題の効率的な解決に役立つ重要なリソースが集約されています。

## トラッキングの基本を学ぶ

モニタリングとトラブルシューティングに取り組む前に、トラッキングに関する強固な基盤を持っていることを確認します。 Journey Optimizerは、すべてのチャネルにわたって、顧客インタラクション、配信パフォーマンス、システムヘルスをキャプチャする包括的なトラッキング機能を提供します。

カスタム指標の設定方法、URL パラメーターの設定方法、組み込みレポートを使用したトラッキングデータへのアクセス方法など、メール、web エクスペリエンス、プッシュ通知、アプリ内メッセージなどにトラッキングを実装する方法について説明します。 [&#x200B; トラッキングの概要 &#x200B;](../using/start/get-started-tracking.md)

## レポートと監視

主要なレポートにアクセスして、ジャーニーのパフォーマンスを監視し、指標を分析し、問題を特定します。

* [ジャーニーグローバルレポート &#x200B;](../using/reports/journey-global-report-cja.md) – すべてのチャネルにわたる包括的なジャーニーパフォーマンス指標および分析を表示します。

* [&#x200B; ライブレポートのジャーニー](../using/reports/journey-live-report.md) - リアルタイムのジャーニー実行データを監視し、アクティブなカスタマーインタラクションを追跡します。

* [ジャーニーステップイベント &#x200B;](../using/reports/journey-step-events-overview.md) - トラブルシューティングと最適化のために、詳細なステップバイステップのジャーニー実行イベントを分析します。

* [&#x200B; レポートの基本を学ぶ &#x200B;](../using/reports/report-gs-cja.md) - レポート機能、指標およびジャーニーインサイトへのアクセス方法について説明します。

* [&#x200B; 配信品質の監視 &#x200B;](../using/reports/deliverability.md) - メッセージ配信率を追跡し、チャネル全体で潜在的な配信の問題を特定します。
