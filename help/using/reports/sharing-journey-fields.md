# ジャーニーフィールド{#sharing-journey-fields}

![](../assets/do-not-localize/badge.png)

このミックスインは、ジャーニースキーマで（jearryStepEventとの関連で）使用されます。 これには、次のフィールドが含まれます。

## jurneryID

メインジャーニーのID。

型：文字列

## jeurneyVersionID

ジャーニーバージョンのID。 このIDは、ジャーニーのIDを表します。

型：文字列

## name

ジャーニーの名前。

型：文字列

## 説明

ジャーニーの説明。

型：文字列

## version

バージョン。`major`.`minor`で表します。

型：文字列
