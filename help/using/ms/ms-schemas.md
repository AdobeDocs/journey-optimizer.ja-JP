# リレーショナルスキーマの作成 {#ms-schemas}

スキーマは、データの構造と形式を表し、検証します。現実のオブジェクト (人物など) の概念上の定義を提供し、そのオブジェクトの各インスタンスに含める必要があるデータ (名前や誕生日など) の概要を説明します。
