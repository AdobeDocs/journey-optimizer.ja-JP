# AI アシスタントプロンプトのベストプラクティス {#ai-assistant-prompting-guide}

>[!CONTEXTUALHELP]
>id="ajo_ai_assistant_prompt"
>title="プロンプトの例"
>abstract="高いコンバージョン率につながる、ブランドに即したマーケティングコンテンツを生成する効果的なプロンプトの作成方法について詳しくは、Journey Optimizer ドキュメントを参照してください。"

このガイドは、リクエストを構造化し、明確に意図を伝え、AI がブランドガイドライン、オーディエンスのニーズ、キャンペーンの目標に沿ったメッセージを生成できるようにするのに役立ちます。
AI アシスタントが目標に合わせてカスタマイズされた高品質でブランドに即したマーケティングコンテンツを生成できるようにする効果的なプロンプトを書き込む方法について説明します。

## CO-STAR フレームワークの使用 {#costar-framework}

AI アシスタントで最適な結果を得るには、CO-STAR フレームワークを使用してプロンプトを整理します。この構造化されたアプローチにより、AI がユーザーのニーズを正確に理解できます。

| コンポーネント | 意味 | これが重要な理由 |
|-|-|-|
| **C - コンテキスト** | キャンペーン、製品または状況に関する背景 | AI が全体像を把握するのに役立ちます |
| **O - 目標** | 特定のマーケティング目標 | コンテンツが達成すべき目標を推進します |
| **S - スタイル** | コミュニケーション方法 | アプローチを設定します |
| **T - トーン** | 感情的なスタイルと声 | メッセージの雰囲気を形作ります |
| **A - オーディエンス** | ターゲティングするオーディエンス | メッセージが適切な人物の共感を得られるようにします |
| **R - 要件** | 特定の制約または必須 | 境界と重要な要素を定義します |

### 品質チェックリスト {#quality-checklist}

コンテンツを生成する前に、次の点を確認します。

&amp;check; **明確な目標**：アクション、製品／サービス、価値、コンテキストを明確に示します。

&amp;check; **定義済みのターゲットオーディエンス**：デモグラフィック、役割、セグメントを指定します。

&amp;check; **コンテンツタイプ一致**：目標は選択したチャネルまたは形式と一致します。

&amp;check; **設定済みのドロップダウンの選択**：トーン、戦略、ロケールが選択されますが、プロンプトには含めません。

&amp;check; **指定済みのドキュメントの焦点**：参照するコンテンツまたはセクションをハイライト表示します。

&amp;check; **ブランド一致**：適切なブランドガイドラインが選択されます。

&amp;check; **現実的な範囲**：レイアウトの変更、スタイル設定、構造の編集のリクエストを回避します。

## 効果的なマーケティング目標の書き込み {#marketing-objectives}

### 具体的でアクション指向であること

マーケティング目標を作成する際は、明確で実用的かつ測定可能であることを確認します。あいまいなステートメントや汎用ステートメントは回避します。

**優れた目標の例：**

&amp;check; 「AI を活用した新しい分析ダッシュボードの 30 日間無料体験版への新規登録を推進します」

&amp;check; 「3月15日（PT）開催の「クラウドコストを 40％削減」に関する B2B ウェビナーのリードを生成します」

&amp;check; 「12月25日（PT）までの期間限定で、プレミアムサブスクリプションを 25％のホリデーディスカウントで宣伝します」

**回避すべき例：**

✕ 「製品を宣伝します」（あいまいすぎる）

✕ 「商品を購入してもらいます」（不明確な値）

✕ 「新機能に関するメールを送信します」（目的が不明瞭）

### 目標の構造化

AI が関連性の高いコンテンツを生成できるように、常にコンテキストと価値提案を提供します。
効果的な目標を書き込むには、次の式を使用します。**アクション + 製品／サービス + 価値／メリット + 緊急性／コンテキスト**

**優れた目標の例：**

&amp;check; 「持続可能な生活習慣を追跡し、パーソナライズされた環境に優しいレコメンデーションを提供する当社の新しいモバイルアプリのダウンロードを促進します」

&amp;check; 「マーケター向けの高度なデータビジュアライゼーション手法に関する専用ワークショップへの登録を宣伝します」

&amp;check; 「週 5 時間以上を節約できる革新的な AI ライティングアシスタントを紹介する製品ローンチイベントへの参加者を増やします」

**回避すべき例：**

✕ 「新しいアプリを発表します」（価値提案とコンテキストが欠落している）

✕ 「ワークショップに新規登録する人物を増やします」（オーディエンスとメリットに関する特異性が欠落している）

✕ 「イベントを宣伝します」（明確なアクション、価値、緊急性が欠落している）

## 新しいコンテンツの作成と既存のコンテンツの変更の比較 {#new-vs-modify}

リクエストに新しいコンテンツの生成が含まれるか、既存するコンテンツの更新が含まれるかを明確に示します。この区別は、AI が適切なアプローチを選択する際のガイドとなり、より正確で有用な結果を確保するために重要です。

### 新しいコンテンツの作成

この戦略は、マーケティングキャンペーンを開始する際、新製品を発表する際、あらゆる種類の更新または刷新されたコミュニケーションを開始する際に適用します。これにより、メッセージの開始が強力になり、目標と一致するようになります。

**プロンプトの方法** ➤ 新しいコンテンツを作成する際は、既存のコンテンツを参照せずにマーケティング目標に焦点を当てます。

>[!BEGINSHADEBOX]

**例：**

* **SaaS 体験版**：「中小企業経営者向けに新しい CRM ソフトウェアをローンチし、50％の時間節約、90％のリードの選定の高速化を強調し、パーソナライズされたオンボーディングを備えた 30 日間の無料体験版を提供します」
* **機能の発表**：「技術に関する意思決定者をターゲットに、エンタープライズ顧客の開発時間を 60％短縮する新しい API 統合機能を発表します」
* **イベントのプロモーション**：「Google、Meta、Adobe のエキスパートが参加し、実用的なインサイトと限定席（最大 500 名）を強調した「Digital Marketing Trends 2024」に関するウェビナーへの登録を推進します」

>[!ENDSHADEBOX]

### 既存のコンテンツの変更

>[!TIP]
>
>詳しく説明、要約、簡素化などの標準的な変更の場合は、カスタムプロンプトを書き込む代わりに&#x200B;**絞り込み**&#x200B;機能を使用します。[詳細情報](generative-uc.md##refine)

現在のマーケティングキャンペーンを更新、刷新、適応させる必要がある場合に適用します。この方法では、増分的な改善がサポートされるので、ゼロから開始することなく、メッセージの関連性を維持できます。

**プロンプトの方法** ➤ 既存のコンテンツを変更する際は、変更する内容と変更方法を明確に指定します。

>[!BEGINSHADEBOX]

**例：**

* **キャンペーンの刷新**：「製品ローンチに関するメールを変更し、一般的な生産性向上のメリットの代わりに、エンタープライズセキュリティ機能に焦点を当てて、コンプライアンス認定資格を持つ IT 意思決定者をターゲットにします」
* **オーディエンスの基点**：「ソフトウェアデモの招待状を医療に特化したものに調整し、一般的な例を医療のユースケースと HIPAA コンプライアンスのメリットに置き換えます」
* **季節の最新情報**：「第 3 四半期のプロモーションキャンペーンを第 4 四半期のホリデーショッピングに合わせて更新し、新学期の買い物からホリデーギフトの贈り物に焦点を移し、迅速な発送を重視します」

>[!ENDSHADEBOX]

## 詳細設定によるプロンプトの強化 {#personalize-prompt}

### コミュニケーション戦略のタイプ

>[!TIP]
>
>特殊な処理のプロンプトで指定する代わりに、テキスト設定メニューのコミュニケーション戦略ドロップダウンを使用します。

コミュニケーション戦略では、メッセージの影響力を最大化するために、どのように伝えるかを決定します。緊急性を構築する、信頼を確立する、迅速な行動を促すなど、目標によって効果的な戦略は異なります。キャンペーンの目標とオーディエンスのマインドセットに最も一致する戦略を選択します。

| **戦略** | **最適な対象** | **メッセージスタイル** | **例** |
|--------------|--------------|------------------------|--------------|
| **至急** | 時間的制約が厳しいオファー、期限、即時アクション | 時間ベースの言語で即座にプレッシャーを作成 | 「今すぐお申し込みください：オファーは午前 0 時に終了します」<br>「空き枠がなくなる前に今すぐご登録ください」<br>「48 時間限定のフラッシュセールが開始します」 |
| **FOMO（取り残されることへの恐れ）** | 制限付きのオファー、イベント、排他的アクセス | 緊急性、希少性、時間的プレッシャーの手がかりを使用 | 「残りわずか 24 時間。限定在庫で 40％オフ」<br>「ラストチャンス：早期割引は明日まで」<br>「ベータ版の残り枠は 100 名様のみ」 |
| **ソーシャルプルーフ** | 信頼構築、お客様の声、人気商品 | 他のユーザーのエクスペリエンスと検証を活用 | 「50,000 人以上の満足したお客様に加わりましょう」<br>「業界の専門家から 5 点中 4.9 の評価」<br>「Fortune 500 企業から信頼されています」 |
| **希少性** | 制限付きの在庫、排他的リリース、需要の高い商品 | 限定提供と排他性を強調 | 「在庫残り 5 個のみ」<br>「限定版：100 個のみ」<br>「限定リリース：先着順」 |
| **インセンティブ** | プロモーション、報酬プログラム、特別オファー | 目に見えるメリットと報酬を強調 | 「初回注文が 20％オフ」<br>「今週末はポイント 2 倍」<br>「50 ドル以上のご注文で送料無料」 |
| **排他性** | プレミアム商品、VIP エクスペリエンス、メンバー限定オファー | プレミアムな位置付け言語と特別なアクセス言語を使用 | 「プライベートプレビューへの限定ご招待」<br>「エリートメンバーシップで高度な分析機能をご利用いただけます」<br>「現在当社の製品を利用する Fortune 500 企業に加わりましょう」 |
| **ゲーミフィケーション** | エンゲージメントキャンペーン、ロイヤルティプログラム、インタラクティブコンテンツ | ゲームの仕組みと達成言語を使用 | 「次の報酬レベルをロック解除しましょう」<br>「チャレンジを完了してバッジを獲得しましょう」<br>「リーダーボードを駆け上がって限定賞品を獲得しましょう」 |
| **情報** | 教育、研究、複雑な製品、思想的リーダーシップ | 事実、データ、インサイト、説明を使用 | 「10,000 件以上のインタラクション分析から得られた 5 つのインサイト」<br>「新しい研究で 3 つの画期的なアプローチが明らかに」<br>「マーケティングにおける AI 導入完全ガイド」 |
| **教育とインサイト** | 学習コンテンツ、業界トレンド、ベストプラクティス | 貴重な知識と実用的なポイントを提供 | 「エキスパートガイドで高度なテクニックを習得します」<br>「トップマーケターが実践する 7 つの戦略を確認します」<br>「5 つのステップでワークフローを最適化する方法について説明します」 |

### 適切なトーンの設定 {#tone}

>[!TIP]
>
>プロンプトで指定する代わりに、テキスト設定メニューのトーンドロップダウンを使用します。

トーンは、オーディエンスがメッセージをどのように認識し、応答するかを形作ります。ブランドの声やプロファイルジャーニーのステージに一致するトーンを常に選択します。

次の表を使用して、各トーンの詳細を確認します。最適なタイミングや、各スタイルに適したコンテンツの例も掲載しています。

| トーン性 | 最適な対象 | コンテンツの例 |
|-|-|-|
| プロフェッショナル | B2B コミュニケーション、正式発表 | 「戦略的パートナーシップを発表できることを嬉しく思います…」 |
| 共感的 | カスタマーサポート、機密性のあるトピック | 「これがどれほどイライラすることかを理解しています…」 |
| ユーモラス | 魅力的なキャンペーン、わかりやすいコンテンツ | 「警告：生産性が大幅に向上する可能性があります。」 |
| 刺激的 | 製品ローンチ、イベントプロモーション | 「これが待ち望んでいた瞬間です。」 |
| 感動的 | やる気を起こさせるキャンペーン、ブランドの目的 | 「力を合わせれば、世界を変えることができます…」 |
| 説得性 | セールスキャンペーン、コンバージョン率 | 「この期間限定のチャンスをお見逃しなく…」 |
| フレンドリー | 顧客エンゲージメント、ウェルカムメッセージ | 「一緒にここに来てくれて本当に嬉しいです。」 |
| 正式 | 法的コミュニケーション、公式通知 | 「以下の変更についてお知らせいたします…」 |
| 謝罪 | サービスの復元、問題解決 | 「ご不便をおかけしましたことを心よりお詫び申し上げます...」 |
| 断定的 | リーダーシップコンテンツ、権威あるメッセージ | 「今すぐ実行する必要があるのは次のとおりです…」 |
| ストーリーテリング | ブランドナラティブ、感情的なつながり | 「すべてはシンプルな疑問から始まりました…」 |
| 対話的 | 育成キャンペーン、関係の構築 | 「これがどのように役立つかについて説明しましょう…」 |

### ブランドアセットの最適化 {#brand-assets}

>[!TIP]
>
>**ブランドアセット**&#x200B;メニューから既にブランドアセットをアップロードしている場合は、プロンプトで参照する必要はありません。選択したドキュメントが自動的に使用されます。

ブランドアセットは、生成されたコンテンツを特定の正確な詳細情報で充実させる事実に基づく情報を提供します。
製品パンフレットなどの広範なドキュメントをアップロードする場合は、プロンプトに対して焦点を当てる部分を追加します。

* ****_「製品パンフレットを使用します」_**の代わりに**_「高度なセキュリティ機能とコンプライアンス認定、特に SOC 2 コンプライアンスとデータ暗号化に焦点を当てます」を使用する必要があります_

* ****_「ケーススタディを参照します」_**の代わりに**_「医療クライアントの ROI 結果、特に地域医療センターにおける 40％のコスト削減を強調します」を使用する必要があります_

* ****_「技術的な詳細を含めます」_**の代わりに**_「REST API エンドポイントと 99.9％の稼働率 SLA に焦点を当て、API 統合機能と開発者のメリットを強調します」を使用する必要があります_

### コンテンツの絞り込み

>[!TIP]
>
>既存のコンテンツに対して詳しく説明、要約、簡素化、トーンを変更、翻訳する場合は、プロンプトの代わりに絞り込み機能を使用します。[詳細情報](generative-uc.md#refine)

コンテンツが生成されたら、**絞り込み**&#x200B;機能を使用して、次のオプションでコンテンツを繰り返し処理および強化します。

| **アクション** | **ユースケース** | **プロンプトの例** |
|-|-|-|
| **詳しく説明** | 簡潔なコンテンツに深みと詳細を追加 | 「セキュリティ機能の技術的なメリットについて詳しく説明してください」 |
| **要約** | 長いコンテンツを様々な形式に圧縮 | 「この製品の概要をソーシャルメディアへの投稿用に要約してください」 |
| **簡素化** | 複雑なコンテンツをよりアクセスしやすくする | 「この技術的な説明を一般的なオーディエンス向けに簡素化してください」 |
| **トーンを変更** | 様々なオーディエンスに合わせてコンテンツを調整 | 「若い世代のデモグラフィック向けに、よりカジュアルなトーンに変更してください」 |
| **トランスクリエイト** | 翻訳を超えた文化的適応 | 「このキャンペーンを日本マーケット向けにトランスクリエイトしてください」 |
