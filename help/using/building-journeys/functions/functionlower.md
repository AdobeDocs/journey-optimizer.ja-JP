# 古い {#lower}

パラメーターの小文字のバージョンを返します。

## 項目

値

## 関数のシンタックス

`lower(<parameter>)`

## 指定

* 値

## シグネチャと戻り値の型

`lower(<string>)`

ストリングを返します。

## 一

`lower("A")`

「A」を返します。
