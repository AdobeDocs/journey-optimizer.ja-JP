# lower {#lower}

パラメーターを小文字にしたものを返します。

## カテゴリ

文字列

## 関数の構文

`lower(<parameter>)`

## パラメーター

* 文字列

## シグネチャと戻り値のタイプ

`lower(<string>)`

文字列を返します。

## 例

`lower("A")`

「a」を返します。
