# upper {#upper}

パラメーターを大文字にしたものを返します。

## カテゴリ

文字列

## 関数の構文

`upper(<parameters>)`

## シグネチャと戻り値のタイプ

`upper(<string>)`

文字列を返します。

## 例

`upper("b")`

「B」を返します。
