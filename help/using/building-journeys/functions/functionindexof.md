# indexOf {#indexOf}

最初の引数のストリングが特定のストリング (2 番目の引数ストリング) で終わっているかどうかをチェックします。 ストリングが見つからない場合は、-1 を返します。

## 項目

値

## 関数のシンタックス

`indexOf(<parameters>)`

## パラメーター

| 指定 | 入力 |
|-----------|------------------|
| 値 | 値 |
| 指定された値 | 値 |

## シグネチャと戻り値の型

`indexOf(<string>,<string>)`

整数値を返します。

## 一

`indexOf("Hello", "l")`

2を返します。

説明

「Hello」の場合、最初に出現する &quot;l&quot; は2の位置になります。
