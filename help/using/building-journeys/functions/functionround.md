# 数値 {#round}

引数に最も近い整数値を返します。これは、正の無限大に関連付けます。

## 項目

数学

## 関数のシンタックス

`round(<parameters>)`

## パラメーター

* 形式
* 整数

## シグネチャと戻り値の型

`round(<decimal>)`

`round(<integer>)`

整数を返します。

## 一

`round(3.14)`

3を返します。

`round(3.54)`

4を返します。

`round(-3.14)`

-3 を返します。

`round(3)`

3を返します。
