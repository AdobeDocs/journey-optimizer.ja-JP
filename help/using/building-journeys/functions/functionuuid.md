# uuid {#uuid}

ランダムな UUID(Universal Unique IDentifier) を生成します。

## カテゴリ

文字列

## 関数の構文

`uuid()`

## パラメーター

この関数にはパラメーターは必要ありません。

## 署名と戻り値の型

`uuid()`

文字列を返します。

## 例

`uuid()`

&quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;を返します。
