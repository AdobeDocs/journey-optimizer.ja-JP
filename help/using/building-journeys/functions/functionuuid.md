# uuid {#uuid}

ランダムな UUID（Universal Unique IDentifier）を生成します。

## カテゴリ

文字列

## 関数の構文

`uuid()`

## パラメーター

この関数にはパラメーターは必要ありません。

## シグネチャと戻り値のタイプ

`uuid()`

文字列を返します。

## 例

`uuid()`

「79e70b7f-8a85-400b-97a1-9f9826121553」を返します。
