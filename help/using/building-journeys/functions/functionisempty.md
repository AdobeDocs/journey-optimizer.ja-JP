# isEmpty {#isEmpty}

パラメーター内の文字列に文字が含まれていない場合は true を返します。

## カテゴリ

文字列

## 関数の構文

`isEmpty(<parameters>)`

## パラメーター

* 文字列

## シグネチャと戻り値のタイプ

`isEmpty(<string>)`

ブール値を返します。

## 例

`isEmpty("")`

true を返します。

`isEmpty("Hello World")`

false を返します。
