# 裁断 {#trim}

開始および終了の空白が削除されます。

## 項目

値

## 関数のシンタックス

`trim(<parameters>)`

## 指定

| 指定 | 入力 |
|-----------|------------------|
| 値 | 値 |

## シグネチャと戻り値の型

`trim(<string>)`

ストリングを返します。

## 一

`trim(" Hello ")`

は「Hello」を返します。
