# currentTimeInMillis {#currentTimeInMillis}

現在の時刻をエポックミリ秒単位で返します。

## カテゴリ

日付

## 関数の構文

`currentTimeInMillis()`

## パラメーター

この関数にパラメーターはありません。

## シグネチャと戻り値のタイプ

`currentTimeInMillis()`

整数を返します。

## 例

`currentTimeInMillis()`

「1544712617131」を返します。
