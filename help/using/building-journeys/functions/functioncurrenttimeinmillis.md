# currentTimeInMillis {#currentTimeInMillis}

現在の時刻をエポックミリ秒単位で返します。

## 項目

古い

## 関数のシンタックス

`currentTimeInMillis()`

## パラメーター

この関数は、パラメーターを使用しません。

## シグネチャと戻り値の型

`currentTimeInMillis()`

整数値を返します。

## 例

`currentTimeInMillis()`

「1544712617131」を返します。
