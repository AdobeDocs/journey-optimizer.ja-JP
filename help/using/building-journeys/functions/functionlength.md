# 長さ {#length}

パラメーターのストリング式の文字数を返します。

## 項目

値

## 関数のシンタックス

`length(<parameters>)`

## 指定

* 値

## シグネチャと戻り値の型

`length(<string>)`

整数値を返します。

## 一

`length("Hello World")`

11を返します。
