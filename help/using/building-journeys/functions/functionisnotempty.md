# isNotEmpty {#isNotEmpty}

パラメーターのストリングが空でない場合は、true を返します。

## 項目

値

## 関数のシンタックス

`isNotEmpty(<parameters>)`

## パラメーター

* 値

## シグネチャと戻り値の型

`isNotEmpty(<string>)`

ブール値を返します。

## 一

`isNotEmpty("")`

False を返します。

`isNotEmpty("hello")`

True を返します。
