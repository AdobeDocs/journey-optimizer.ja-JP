# isNotEmpty {#isNotEmpty}

パラメーター内の文字列が空でない場合は true を返します。

## カテゴリ

文字列

## 関数の構文

`isNotEmpty(<parameters>)`

## パラメーター

* 文字列

## 署名と戻り値の型

`isNotEmpty(<string>)`

ブール値を返します。

## 例

`isNotEmpty("")`

false を返します。

`isNotEmpty("hello")`

true を返します。
